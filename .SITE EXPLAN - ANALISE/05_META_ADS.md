## CAMPANHA 1 — TRÁFEGO / RECONHECIMENTO

**Objetivo:** Levar pessoas qualificadas para o site
**Orçamento:** R$ 300–500/mês
**Veiculação:** Feed Instagram + Stories

### Público 1A — Interesses
```
Localização: Goiânia, GO + 30km
Idade: 28–55 anos
Sexo: Todos
Interesses:
  → Decoração de interiores
  → Arquitetura e design
  → Casa própria / Imóveis
  → Reforma e construção
  → Marcas: MRV, Cyrela, Viva Real (compradores de imóvel)
Excluir: já enviaram mensagem / já converteram
```

### Público 1B — Lookalike
```
Base: Lista de clientes (carregar no Gerenciador)
Lookalike: 1–3% · Brasil · Filtro por Goiânia
```

### Criativos para Campanha 1

**Criativo A — Portfólio (Carrossel)**
```
Slide 1: Foto impactante de cozinha. Texto: "Sua cozinha pode ser assim."
Slide 2: Closet organizado. Texto: "Do sonho ao ambiente real."
Slide 3: Processo 3D. Texto: "Você aprova antes de fabricarmos."
Slide 4: Logo + CTA. Texto: "Projeto 3D grátis. Showroom em Goiânia."
Legenda: "Móveis planejados com garantia de até 5 anos. Projeto 3D grátis.
         Solicite seu orçamento sem compromisso. 📍 Goiânia"
CTA: "Saiba mais" → site
```

**Criativo B — Reels (Vídeo 15s)**
```
Roteiro:
0–3s: Hook visual — ambiente incrível revelado
3–10s: Texto na tela: "Projeto 3D grátis + Garantia até 5 anos"
10–15s: Logo Explan + "Showroom no Setor Bueno, Goiânia"
CTA: "Solicitar orçamento"
```

## CAMPANHA 2 — GERAÇÃO DE LEADS

**Objetivo:** Capturar dados de quem já conhece a Explan
**Orçamento:** R$ 200–400/mês
**Formato:** Lead Ads (formulário nativo Meta) + Stories

### Formulário de Lead (configurar no Meta)
```
Título: "Solicite seu Projeto 3D Grátis"
Perguntas:
  1. Nome completo (auto-preenche)
  2. WhatsApp (auto-preenche)
  3. Qual ambiente você quer transformar?
     ○ Cozinha  ○ Closet  ○ Dormitório  ○ Banheiro  ○ Corporativo
  4. Você tem imóvel em Goiânia?
     ○ Sim, novo  ○ Sim, em reforma  ○ Estou planejando
Botão: "Solicitar Projeto Grátis"
Mensagem pós-envio: "Obrigado! Nossa equipe entrará em contato em até 2h."
```

### Criativos para Campanha 2

**Criativo — Oferta do Projeto 3D**
```
Imagem: Tela de software 3D mostrando cozinha planejada
Título: "Projeto 3D do seu ambiente — Grátis"
Texto: "Veja como ficará seu apartamento antes de fabricar
        um único painel. Sem compromisso. ✓ Atendemos Goiânia e região."
CTA: "Cadastrar"
```

## CAMPANHA 3 — MENSAGENS / CONVERSÃO

**Objetivo:** Gerar contato direto no WhatsApp
**Orçamento:** R$ 300–500/mês
**Formato:** Anúncios "Click to WhatsApp"

### Público 3A — Retargeting Quente
```
Visitaram o site nos últimos 7 dias (Pixel instalado)
OU
Engajaram no Instagram nos últimos 30 dias
OU
Visualizaram vídeos em 50%+ nos últimos 14 dias
```

### Mensagem pré-preenchida no WhatsApp
```
"Olá! Vi a Explan no Instagram e quero saber mais sobre
[cozinha / closet / dormitório] planejado. Podem me atender?"
```

### Criativos para Campanha 3

**Criativo A — Depoimento + CTA**
```
Formato: Post simples ou Stories
Fundo: Foto do projeto do cliente
Texto sobreposto: "A Ana amou o resultado. Veja o projeto dela →"
Estrelas: ★★★★★
CTA: "Falar no WhatsApp"
```

**Criativo B — Urgência Sazonal**
```
Texto: "📅 Agenda de [mês] quase cheia.
        Garanta seu projeto antes que a fila feche.
        ✓ Projeto 3D grátis  ✓ Goiânia"
CTA: "Solicitar agora"
```

## PIXEL META — INSTALAÇÃO E EVENTOS

Instalar Pixel Meta no site WordPress (via plugin PixelYourSite ou manualmente):

| Evento | Gatilho | Importância |
|---|---|---|
| `PageView` | Toda página | Básico |
| `ViewContent` | Ver galeria / portfólio | Retargeting |
| `Lead` | Envio do formulário | Conversão |
| `Contact` | Clique no WhatsApp | Conversão principal |
| `InitiateCheckout` | Clique em "Solicite Orçamento" | Micro-conversão |

## REGRAS DE GESTÃO

- Testar no mínimo 2 criativos por conjunto de anúncios
- Pausar criativos com CTR < 1% após 3 dias
- Escalar (aumentar orçamento 20%) criativos com CPC < R$ 1,50
- Renovar criativos a cada 3–4 semanas (fadiga de audiência)
- Analisar relatórios toda segunda-feira
