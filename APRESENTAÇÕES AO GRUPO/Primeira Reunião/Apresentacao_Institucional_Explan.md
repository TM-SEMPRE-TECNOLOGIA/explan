## IDENTIDADE VISUAL (aplicar em todos os slides)

### Paleta de Cores
| Elemento | Cor | Uso |
|---|---|---|
| Primária | `#2D5016` (verde musgo / natural) | Títulos, destaques, ícones |
| Fundo principal | `#FFFFFF` | Background limpo |
| Fundo alternativo | `#F5F4F0` | Seções de apoio |
| Texto principal | `#1A1A1A` | Corpo de texto |
| Texto secundário | `#6B6B6B` | Subtítulos, legendas |
| Acento CTA | `#3D7A1A` | Bordas, linhas divisórias |

### Tipografia
- **Títulos (H1):** Sans-serif moderna — Montserrat Bold ou Inter Bold — 36–48px
- **Subtítulos (H2):** Montserrat SemiBold — 24–28px
- **Corpo de texto:** Inter Regular — 16–18px
- **Destaque / tag:** Montserrat Medium, uppercase com letter-spacing — 12–14px
- **Hierarquia:** bold para nomes/cargos, regular para descrição

### Estilo Visual
- Minimalista, sofisticado, muito espaço em branco
- Fotografias de alta resolução como elementos visuais principais
- Ícones geométricos simples (linha fina, cor primária)
- Divisórias sutis (linha 1px na cor primária)
- Layout assimétrico com blocos de conteúdo à esquerda e imagem à direita (regra 60/40)
- Margens generosas (mínimo 64px laterais)

### Logo
- Usar logo Explan no canto superior esquerdo em todos os slides
- Versão cor (verde + texto escuro) sobre fundo claro
- Versão branca sobre slides com fundo escuro/imagem

## NOTAS PARA O MANUS

1. **Versões individuais:** Gere 4 versões — uma para cada nome (Mayron, Emerson, João) e uma versão geral para toda a equipe. Nos slides de apresentação individual, o slide do colaborador em questão deve ter destaque especial (borda colorida ou ícone destacado).

2. **Consistência visual:** Mantenha o mesmo espaçamento, margens (64px laterais) e hierarquia tipográfica em todos os slides.

3. **Transições sugeridas:** Fade suave (0.3s) entre slides. Sem efeitos excessivos.

4. **Resolução de exportação:** 1920×1080px mínimo para apresentação em tela; 300 DPI se for imprimir.

5. **Ícones:** Usar estilo de linha fina (stroke), cor `#2D5016`, sem preenchimento sólido.

6. **Fontes seguras (fallback):** Se Montserrat ou Inter não estiverem disponíveis no Manus, usar `Arial` para corpo e `Georgia` para títulos.
