## CONFIGURAÇÕES TÉCNICAS ESSENCIAIS

### Conversões para rastrear (Google Tag Manager)
```
1. Clique em "Solicite Orçamento" (WhatsApp)
   → Evento: whatsapp_click
   → Valor: R$ 80 (CPL médio estimado)

2. Envio do formulário de contato
   → Evento: form_submit
   → Valor: R$ 80

3. Ligação telefônica (via extensão de chamada)
   → Evento: phone_call
   → Valor: R$ 60

4. Clique no endereço (intenção de visitar showroom)
   → Evento: location_click
   → Valor: R$ 40
```

### Quality Score — Melhorar para reduzir CPC
- Relevância do anúncio com a palavra-chave: cada grupo tem anúncio específico ✓
- CTR esperado: usar palavras de ação no título (Solicite, Veja, Garanta)
- Experiência na página de destino: implementar melhorias do arquivo UI/UX ✓
